Proyectos

"Los proyectos que verán a continuación son resultado de mi último bootcamp enfocado en el desarrollo de aplicaciones con Python y el Framework Django."

Proyecto Digidesk

El proyecto Digidesk se centra en sincronizar y guardar los digimones en la página https://digidesk-concha-david.vercel.app/. Para lograr esto, se emplea el framework Django en combinación con la API Digimons. La aplicación se encarga de acceder a la API, obtener la información sobre los digimones y guardarla en la base de datos PostgrestSql.

Proyecto Todos

Proyecto Todolist es una aplicación basada en Django que posibilita a los usuarios registrar, marcar como completadas y eliminar actividades en una página web.
Proyecto Registro Clínico Electrónico Deportivo

El Proyecto Registro Clínico Electrónico es una aplicación desarrollada con Django que permite brindar atención médica especializada y centralizada a los deportistas, enfocándose desde el punto de vista deportivo. Con esta plataforma, es posible generar registros clínicos electrónicos y gestionar el seguimiento y tratamiento de los deportistas de manera eficiente.
